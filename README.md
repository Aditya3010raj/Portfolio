## 💎 Key Features
- **The Portfolio:** Dynamic showcase of engineering and software projects fetching data via **GROQ**.
- **The Technical Blog:** Built for documentation with syntax highlighting and rich-text rendering via **Sanity.io**.
- **Headless CMS:** Decoupled architecture allows content updates (projects/blogs) without code redeployment.
- **Animations:** Fluid transitions and component entrances powered by **Framer Motion**.

## 🛠️ Tech Stack
- **Framework:** Next.js 14
- **CMS:** Sanity.io (Headless CMS)
- **Styling:** Tailwind CSS
- **Interactivity:** Framer Motion
- **Deployment:** Vercel

## 📂 Content Schema (Sanity)
The backend is structured for granular control:
- **Project Schema:** Includes Title, Tech Stack (Array), and GitHub/Live Links.
- **Blog Schema:** Optimized for technical writing using Sanity's Portable Text.
- **Image Pipeline:** Uses Sanity's CDN for automated WebP conversion and responsive resizing.

## 📦 Quick Start
1. **Clone:** `git clone https://github.com/Aditya3010raj/portfolio.git`
2. **Install:** `npm install`
3. **Environment:** Setup `.env.local` with your `NEXT_PUBLIC_SANITY_PROJECT_ID`.
4. **Dev Mode:** `npm run dev`
